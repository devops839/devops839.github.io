AD_ArgoCD_AKS_Project

This pipeline automates the build, scan, and deployment process for microservices in an Azure Kubernetes Service (AKS) cluster using ArgoCD. It includes SonarQube code analysis, Trivy security scans, Docker image builds, and pushes images to Azure Container Registry (ACR). After updating Kubernetes manifests, ArgoCD automatically deploys the application to AKS, with monitoring available through Grafana.

AD_Java_Project

This Azure DevOps pipeline automates the build and deployment of a Java voting application. It installs Java 17, performs SonarQube analysis to check code quality, compiles and packages the application using Maven, and builds a Docker image. The image is then scanned for vulnerabilities using Trivy, and finally, the image is pushed to Azure Container Registry. The pipeline ensures both code quality and security, providing a seamless, automated deployment process to Azure Kubernetes Service (AKS).
